A CHIP-8 interpreter written in Rust.

about this game
This Space Invaders ROM (author: David Winter) comes from the "revival pack" that can be found on the Internet (e.g. here).
help
The CHIP-8 virtual machine requires a hex keyboard that has 16 keys ranging from 0 to F, which isn't trendy these days. For this interpreter, the 16 keys are remapped to those on the top left part of a QWERTY keyboard: 
        1 2 3 C      1 2 3 4
        4 5 6 D  =>  Q W E R
        7 8 9 E      A S D F
        Z X C V      A 0 B F
Start playing Space Invaders with W. Move to the left with Q and to the right with E. Fire with W.

This project started as a "week-end project" by @couac. The source code is available on GitHub (MIT license).
